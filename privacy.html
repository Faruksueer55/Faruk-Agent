Stand: Juni 2025

2. Welche Daten werden verarbeitet

Diese App greift ausschliesslich auf oeffentlich zugaengliche Social Media Insights des Accountinhabers (@faruksueer) zu. Es werden keine personenbezogenen Daten Dritter gespeichert oder verarbeitet.

3. Zweck der Datenverarbeitung

Die erhobenen Daten (Reichweite, Impressionen, Follower-Zahlen) werden ausschliesslich zur persoenlichen Content-Analyse des Accountinhabers verwendet.

4. Weitergabe von Daten

Es erfolgt keine Weitergabe von Daten an Dritte. Alle Daten verbleiben beim Accountinhaber.

5. Drittanbieter

Diese App nutzt die Meta Graph API (Instagram/Facebook) sowie die YouTube Data API. Es gelten die jeweiligen Datenschutzbestimmungen von Meta und Google.

6. Datenspeicherung

Es werden keine Daten dauerhaft gespeichert. Alle API-Abfragen erfolgen in Echtzeit und werden nicht protokolliert.
